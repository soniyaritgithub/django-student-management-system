✅ Delete Student  
✅ Search Student  
✅ Login / Logout System

✅ Bootstrap UI  
✅ MySQL Database  
✅ REST API

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/soniyaritgithub/django-student-management-system.git
```

---

## Go to Project Folder

```bash
cd django-student-management-system
```

---

## Create Virtual Environment

```bash
python -m venv .venv
```

---

## Activate Virtual Environment

### Windows

```bash
.venv\Scripts\activate
```

---

## Install Requirements

```bash
pip install -r requirements.txt
```

---

## Run Server

```bash
python manage.py runserver
```
